Homework 4-1

Q1: Give the SELECT command to answer the question Who will go to the party held in Peking University?

A1: 

    SELECT guest_Name

    FROM guest, party, guest_party

    WHERE party.Party_Num = guest_party.Party_Num

    AND guest_party.Guest_Num = guest.Guest_Num

    AND party.Place = "Peking University"

Q3: Tell the difference between a CGI program and a PHP program executed by a Web Server.

A3: (1) CGI is a kind of interface for clients to invoke programs located on the server side.
On the other hand, PHP programs are not accessible for clients. It is designed totally for the server.
(2) CGI programs are used to submit and process user-input data. PHP programs can be used in various functions, e.g., loading dynamic webpages or managing a database.
