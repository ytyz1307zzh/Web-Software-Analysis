Q1: Re-implement the survey program in Homework 3 in MVC mode.

A1: The survey can be found here.
My MVC mode is implemented using PHP. Three PHP classes are created, namely Controller, Model, and View. View includes methods to display information on the webpage. Model includes functions to manipulate the MySQL database, i.e. to insert, list or delete data. Controller, as its name suggests, controls the actions of Model and View while separating front-end webpage and back-end programs.
Any other PHP script can basically use the interface provided by Controller to perform its desired behavior. It just need to include the source scripts of these classes and initialize instances of them.
