Q1: What are JavaScript browser sniffers?

A1: JavaScript Browser Sniffer is a browser identifier written in JavaScript. It main function is to detect the user's operating system, browser type & version.

Q2: Why are they so commonly used in javascripts?

A2: Because browser detection is a critical part in web programming with JavaScript. Since different browsers provide different support to HTML events as well as DOM objects, performing browser detection helps programmers write compatible codes for different browser.
